# stomp-simple-broker
A simple stomp message broker sample using Spring Framework

### Compile with:
  `mvn clean install`

### Run in tomcat:
  Copy `target/stomp-simple-broker-0.0.1-SNAPSHOT.war` to tomcat webapp directory

### Run in command line:
  `java -jar target/stomp-simple-broker-0.0.1-SNAPSHOT.war`
